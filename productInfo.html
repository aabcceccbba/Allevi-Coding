Learn About Your Product Information

This table contains the print information including files, pressure, crosslinking, resolution, and wellplate.

  • Files contain the filenames of the input and post-processed print GCODE files.
  • Pressure contains the pressures of the first and the second extruders at the time of print.
  • Crosslinking contains the cl_enabled, a boolean that if photocrosslinking was used during this print; cl_duration, which is the duration of photocrosslinking using during this print in ms; cl_intensity, which is the percent intensity of light used in photocrosslinking..
  • Resolution contains layerNum which is the number of layers in this print, and layerHeight which is the Height of each layer in mm.
  • Wellplate, which is wellplate type used for the print.


    • It also contains the print data, which includes live percentage, elasticity, and dead percentage.

  • LivePercent, which is the percent of final print determined to be alive through live/dead imaging.
  • Elasticity, which is the measure of final print structural rigidity measured in kPa.
  • DeadPercent, which is the percent of final print determined to be dead through live/dead imaging.
